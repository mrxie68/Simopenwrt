# 固件简要说明：

这是个平台固件包，仅供个人使用，只支持x86-64！
请注意选择你需要的设备固件！
50M左右只包含：
HomeProxy
MosDNS
Tailscale
DDNS-Go
需要其他请自行修改配置文件编译。
